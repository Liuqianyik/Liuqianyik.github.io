vue-moment时间格式化

2022-08-13

一、参考资料

  1. momentjs官网
  2. momentjs官网中文版使用文档
  3. github：vue-moment

二、vue安装插件及配置

1. 通过npm安装

$ npm install vue-moment

2. 在项目中引入并配置插件

// main.js全局引入
import VueMoment from 'vue-moment'
import moment from 'moment'

// 设置本地时区
moment.locale('zh-cn')
Vue.use(VueMoment, {
  moment
})

3. 使用

一般后端传来的时间以及我们直接用new Date()。getTime()获取到的都是类似1656673259949的的字符串，在前端页面上展示的时间想要符合预期格式就需要用到momentjs插件

3.1. 常见用法：

  1. 星期，年月日时分秒
    <span>{{ someDate | moment("YYYY") }}</span>
<!-- e.g. "2010" -->
<span>{{ someDate | moment("dddd, hA") }}</span>
<!-- e.g. "Sun, 3PM" -->
<span>{{ someDate | moment("dddd, MMMM Do YYYY, h:mm:ss a") }}</span>
<!-- e.g. "Sunday, February 14th 2010, 3:25:50 pm" -->
  2. 计算时间差
    <!-- 计算somedate到当前时间的间隔，输出eg:一个月前 -->
<span>{{ someDate | moment("from", "now") }}</span>
<!-- 不加“now”，效果一样，以下是上面一行的简写版 -->
<span>{{ someDate | moment("from") }}</span>
<!-- 给出参考时间的写法如下，计算 -->
<span>{{ someDate | moment("from", 1656673259949) }}</span>
  3. 更多用法在github官方项目可以看到
