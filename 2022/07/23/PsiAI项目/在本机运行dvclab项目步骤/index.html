在本机运行dvclab项目步骤

2022-07-23

1. 安装git

参考廖雪峰Git

2. 克隆项目到本地

  • 在gitlab或者github找到想clone的项目地址，如https://gitlab.dvclab.com/frontend/DvcAI-fe.git
  • 到你想下载的目录下打开命令行
    git clone https://gitlab.dvclab.com/frontend/DvcAI-fe.git

    3. 下载依赖包: 一开始项目是用yarn构建的

    yarn install

    4. 运行项目

    npm run dev
