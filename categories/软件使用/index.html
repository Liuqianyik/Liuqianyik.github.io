sublime使用方法记录

2022-07-18

之前一直用的是vscode，现在因为导师要求和项目需要，改用sublime，记录一下从零到一的使用过程，留下一些使用技巧供自己以后回顾 一、下载安装直接去官网sublime下载，是免费的 二、配置 设置主题preferences ==&gt; select theme ==&gt; 选择自己想要的主题即可 安装package control菜单栏tools ==&gt; Install Package Control… 安装插件 打开package control快捷键：Shift + Ctrl + P 在搜索框中输入install，选择第一项“Package Control: Install Package”,点击之后等一会会出现package control的搜索框 接下来就可以搜索需要的插件名..
